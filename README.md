**Draft Web App - Responsive Web Interface Design**

Designed an intuitive and visually appealing web application prototype for Draft, focusing on seamless interaction, responsive layout, and modern aesthetics. Created end-to-end in Figma, this project showcases my skills in interface design, layout hierarchy, and user flow structuring for web platforms.

🔹 Project Highlights

Clean & Modern UI: Minimalist design with thoughtful spacing, typography, and visual balance to enhance usability.
Responsive Layout: Components and grid system designed to adapt across various screen sizes & devices.
Interactive Prototype: Clickable screens and transitions to simulate realistic user experience and navigation.
Structured Information Architecture: Logical content organization to guide users through key tasks and features.

🛠 Tools Used: Figma (for wireframes, high-fidelity screens, and interactive prototyping)

🚀 Design Focus

Emphasis on usability & visual clarity
Consistent use of design principles for hierarchy and readability
Built with scalability in mind to support future features and additions
